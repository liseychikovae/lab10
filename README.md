# lab10

Лабораторная работа Лисейчиковой Екатерины

## Отчет по лабораторной работе X

### Тема

Изучение развертывания виртуальной среды на примере Vagrant.

### Цель работы

Цель работы - описать воспроизводимую Linux-виртуальную машину для сборки проекта.

### Используемое окружение

- основной целевой сценарий: Linux;
- язык: C++11;
- система сборки: CMake;
- дополнительные инструменты добавляются по теме конкретной лабораторной;
- персональные токены и временные адреса не сохраняются в репозитории.

### Структура проекта

- `formatter_lib` - базовая статическая библиотека форматирования;
- `formatter_ex_lib` - расширенная библиотека, использующая `formatter`;
- `solver_lib` - библиотека для решения квадратного уравнения;
- `hello_world_application` - демонстрационное приложение;
- `solver_application` - приложение для вывода корней уравнения;
- `tests` - модульные тесты, начиная с lab05;
- `docs` и `scripts` - дополнительные материалы поздних лабораторных.

### Ход выполнения

- добавлен `Vagrantfile` с образом `ubuntu/jammy64`; 
- папка проекта монтируется в `/workspace`; 
- provision-скрипт устанавливает `cmake`, `g++`, `make` и `git`; 
- после запуска VM проект автоматически конфигурируется и собирается;
- инструкция вынесена в `docs/vagrant.md`.

### Команды для проверки

```bash
vagrant up
vagrant ssh
cd /workspace
cmake --build _build
vagrant halt
```

### Домашнее задание

Домашнее задание выполнено внутри этого же репозитория. Подробное описание вынесено в `HOMEWORK.md`.

Кратко по выполнению:

- виртуальная машина отделяет Linux-сборку от локальной системы;
- provision делает окружение повторяемым для преподавателя и студента;
- проект остается в общей папке, поэтому изменения видны и на хосте, и внутри VM;
- Vagrantfile не содержит персональных секретов.

### Результат

Для проекта описана VM, в которой можно выполнить Linux-сборку даже с macOS-хоста.
