# Домашнее задание lab10

Лабораторная работа Лисейчиковой Екатерины

## Что требовалось сделать

Домашнее задание относится к теме лабораторной работы X: изучение развертывания виртуальной среды на примере vagrant.

## Выполненные пункты

- виртуальная машина отделяет Linux-сборку от локальной системы;
- provision делает окружение повторяемым для преподавателя и студента;
- проект остается в общей папке, поэтому изменения видны и на хосте, и внутри VM;
- Vagrantfile не содержит персональных секретов.

## Как проверить домашнее задание

```bash
vagrant up
vagrant ssh
cd /workspace
cmake --build _build
vagrant halt
```

## Пояснение по реализации

Проект оформлен так, чтобы его можно было перенести в Linux-окружение и собрать стандартными командами CMake. Файлы не завязаны на абсолютные пути macOS, персональные секреты не добавлены в git, а дополнительные инструменты подключаются через конфигурационные файлы конкретной лабораторной.
